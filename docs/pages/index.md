# ldap-password-change

A lightweight, self-hosted web service that lets users change their LDAP password through a secure,
modern browser interface — no admin intervention required.

## Features

- **Self-service password change** via a clean, responsive web UI
- **Material Design V3** – Glassmorphism UI with Dark / Light / System theme switching
- **Fully configurable** – YAML-based configuration with environment variable overrides
- **Custom branding** – Inject your own background, favicon, logo, and CSS
- **Structured logging** – JSON request tracing with request IDs and component tags
- **Liveness & Readiness probes** – Kubernetes/Docker-compatible `/health/live` and `/health/ready`
